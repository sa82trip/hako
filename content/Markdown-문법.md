마크다운 기본 문법과 Obsidian 확장을 정리합니다.

## 위키링크 (가장 중요)

다른 페이지로 링크하려면 대괄호 두 개를 씁니다.

```markdown
[[시작하기]]
[[백링크-테스트|별명으로 표시]]
```

이렇게 하면 [[시작하기]]처럼 연결되고, 링크된 페이지에는 백링크가 생깁니다.

## 헤딩

```markdown
# 제목 1
## 제목 2
### 제목 3
```

## 강조

```markdown
**굵게**, *기울임*, `코드`
```

## 목록

```markdown
- 항목 1
- 항목 2
  - 하위 항목
```

## 코드 블록

언어를 지정하면 구문 강조가 적용됩니다.

```python
def hello():
    print("안녕하세요")
```

## 인용

```markdown
> 인용할 내용
```

## 체크리스트

- [x] 완료한 일
- [ ] 해야 할 일
